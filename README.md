# AI Fix (Tampermonkey)

A minimal userscript that copies selected or clicked page content into an AI-friendly format.

## Install

1. Install Tampermonkey or Violentmonkey.
2. Create a new userscript and paste `ai-fix.user.js`.
3. Enable the script.

## Usage

- Click "复制给AI" to enter pick mode.
- Click any element to copy its content.
- If text is selected, it copies the selection immediately.

## Notes

- For iframe content, enable "Inject into all frames" in your userscript manager if possible.
- Cross-origin iframes may be inaccessible due to browser security.
